A secure password generating tool built to help users create strong, random passwords for better online security.

## Features

- Generate cryptographically secure passwords
- Customizable password length
- Option to include/exclude special characters, numbers, uppercase/lowercase letters
- Copy to clipboard functionality
- Clean, user-friendly interface

## Live Demo

**[Try it here](https://secure-password.up.railway.app/)**

## Technologies

- Web application deployed on Railway
- Modern, responsive design
